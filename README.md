## Projenin Amacı
Bu proje, kullanıcıların günlük harcamalarını kolayca kayıt altına almalarını, kategorize etmelerini ve finansal durumlarını basit grafiklerle takip etmelerini sağlayan bir web uygulamasıdır.

## Kullanılan Teknolojiler
* HTML5, CSS3, JavaScript
* Chart.js (Grafik görselleştirme)
* Bootstrap (Arayüz tasarımı)
